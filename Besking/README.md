## 特性

- -多种文章列表布局
- -个人中心类型头部，彰显个人博客特质
- -侧边小工具，可用于展示二维码或qq临时会话
- -自适应手机端，扩大可阅读设备范围
- -聚合首页，又一次与众不同
- -适配模板设置插件，降低使用者技术门槛
- -单独的站内搜索页面
- -不一样的网站数据归档统计
- -遵循typecho模板开发规范
- -保持更新
